# User Management Interface 💻

Frontend dinâmico desenvolvido para interagir com a API de segurança, focado em usabilidade (UX) e tratamento seguro de dados no lado do cliente.

## 🚀 Diferenciais Técnicos
- **Bundler:** Utilização do Parcel para otimização de assets e build rápido.
- **HTTP Client:** Axios para comunicação assíncrona com tratamento de erros.
- **Sessão Segura:** Implementação de persistência de login via Bearer Token armazenado em `localStorage`.
- **Feedback Visual:** Validação de formulários e tratamento de rotas protegidas.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Design Responsivo).
- JavaScript Moderno (ES6+).
- Axios para integração com API.
- DataTables para exibição organizada de informações.

## 🏁 Execução
1. Certifique-se de que o Backend está rodando na porta 3000.
2. Instale as dependências: `yarn`.
3. Inicie o servidor de desenvolvimento: `yarn dev`.
4. O projeto estará disponível em `http://localhost:1234`.
